3. Under "Destination," select **"Save as PDF"**
4. Click **Save**
5. Choose a location and filename (e.g., `ML_vs_Statistical_Models.pdf`)

#### **Option B: Chrome/Edge Print to PDF**
1. Open the file in Chrome or Edge
2. Press **Ctrl+P** or **Cmd+P**
3. Select **"Save as PDF"** from the printer dropdown
4. Configure:
   - **Orientation**: Portrait (recommended)
   - **Paper size**: A4
   - **Margins**: Default or Minimal
5. Click **Save**

#### **Option C: Using a PDF Tool**
- **macOS**: Use built-in Preview → Print → PDF dropdown
- **Linux**: Use `wkhtmltopdf` or `firefox --print-to-file`
- **Windows**: Use dedicated PDF printer (Microsoft Print to PDF)

## 📖 Content Summary

### Core Differences Highlighted

| Aspect | Statistical Model | Machine Learning |
|--------|-------------------|------------------|
| **Goal** | Explain relationships | Predict accurately |
| **Approach** | Theory-driven | Data-driven |
| **Interpretability** | High | Low |
| **Data needed** | 100–10K | 10K–1M+ |
| **Flexibility** | Constrained | Highly flexible |

### Key Analogies

1. **Detective Model**: Statistical model = detective with a theory testing specific hypotheses; ML model = security camera learning patterns
2. **Business Value**: Statistical = "Why did our churn increase?" (insight); ML = "Which customers will churn next?" (action)

### Decision Framework

**Choose Statistical Models when:**
- Need to explain causality
- Regulatory compliance requires transparency
- Small to moderate dataset
- Stakeholders value interpretability

**Choose Machine Learning when:**
- Primary goal is accuracy
- Large datasets available
- Can tolerate black boxes
- Complex non-linear patterns
