Poisson-Gamma GLM Model

Core Formula:

Premium = BaseFreq × BaseSeverity × ∏(Age × Smoking × Exercise × Occupation × Region × PreExist)

Underwriting Engine & Tiering

Decline Rules

  • Age < 18 or > 70
  • Terminal conditions (advanced cancer, organ failure)

Warnings Triggered

  • Cancer (remission), Kidney Disease, Liver Disease
  • Smoker + Cardiac history
  • ≥ 3 pre-existing conditions

Tier Multipliers

  • Silver 1.00× — Baseline
  • Gold 1.45× — Premium
  • Platinum 2.10× — Comprehensive

Admin Pipeline: Model Calibration

Three Phases

Phase 1: Upload Claims CSV

Admin uploads historical claims data (age, smoking, occupation, region, claim_count, claim_amount)

Phase 2: Preview O/E Ratios

System computes Observed vs Expected claim frequencies per coverage type. Shows percentage deltas.

Phase 3: Approve & Deploy

Admin reviews changes, approves calibrated coefficients. New COEFF version goes live in seconds.

Audit Trail: All versions stored in hp_glm_versions table with timestamp, deployed_by, changes_applied

Data Model (PostgreSQL)
